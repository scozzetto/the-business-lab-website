Exporting Data

3 min read

Export your client data as CSV files for reporting, mail merges, external analysis, or backup purposes.

Exporting Client Records

  1. Apply Filters (Optional)
    If you only want to export a subset of clients, apply search terms or column filters first. The export will include only the visible records.
  2. Click Export
    Click the Export button (download icon) in the toolbar above the table.
  3. Choose Columns
    A dialog lets you select which columns to include in the export. By default, all visible columns are selected.
  4. Download the File
    Click Download CSV. The file downloads to your computer's default download location.

What's in the Export

The CSV file contains one row per client with the columns you selected. The first row is headers. The file opens in Excel, Google Sheets, or any spreadsheet app.

Handle exports securely. Client data exports contain personal information. Don't email CSV files unencrypted, and delete exports when no longer needed.
