Client Status & Tags

4 min read

Every client has a status and can have multiple tags. These help you organize your database, segment clients for marketing, and quickly identify key accounts.

Client Statuses

StatusMeaningColor
ActiveCurrent, engaged clientGreen badge
InactiveNo recent visits; may need outreachGray badge
VIPHigh-value or priority clientGold badge

Changing a Client's Status

  1. Open the Client Profile
    Search for and open the client whose status you want to change.
  2. Click the Status Badge
    Near the client's name, click on their current status badge. A dropdown appears with available statuses.
  3. Select the New Status
    Choose the appropriate status. The change is saved automatically and logged.

Manager permission required. Only managers and admins can change client statuses. Front desk staff should request status changes from their manager.

Using Tags

Tags are custom labels you can attach to clients for organization. Unlike status (which has fixed options), tags are flexible and you can create as many as you need.

Common tag examples:

  • Membership: Gold — Identifies membership tier
  • Referral Source: Website — How the client found you
  • Interest: Wellness — Areas of interest
  • Follow-Up Needed — Flags for staff action

Adding Tags to a Client

  1. Open the Client Profile
    Navigate to the client's full profile page.
  2. Find the Tags Section
    Below the client's basic info, find the Tags section with a + Add Tag button.
  3. Add a Tag
    Click + Add Tag, type the tag name, and press Enter. Existing tags appear as suggestions.

To remove a tag, click the × next to it on the client's profile.

Filtering the Database by Tags

Use column filters to find all clients with a specific tag. Go to Filter → Tags → select one or more tags. This is powerful for marketing campaigns and outreach.
