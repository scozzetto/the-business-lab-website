Searching for Clients

4 min read

Finding the right client quickly is essential for daily operations. The database offers instant search, column filters, and advanced query options.

Quick Search

  1. Click the Search Bar
    At the top of the client table, click the search input field. You can also press / to jump to it.
  2. Type Your Query
    Start typing a name, email, phone number, or client ID. Results filter instantly as you type — no need to press Enter.
  3. Click the Result
    Click on the matching client row to open their full profile.

Partial matches work. Typing "joh" will match "John", "Johnson", "Johnny", etc. The search checks first name, last name, email, phone, and client ID simultaneously.

Column Filters

For more targeted searches, use column-specific filters:

  1. Click the Filter Icon
    Next to the search bar, click the Filter button (funnel icon) to reveal filter options.
  2. Select a Field
    Choose which field to filter by: Status, Location, Tags, Date Added, or Balance.
  3. Set Your Criteria
    Select the filter value (e.g., Status = "Active") and the table updates immediately.

You can stack multiple filters. For example: Status = Active AND Location = Downtown.

Clearing Search & Filters

To reset your search: clear the search box and click Clear All Filters (if filters are active). The table will return to showing all clients.

Troubleshooting

Can't find a client I know exists

Check your location filter — the client might be registered at a different location. Try switching to "All Locations" if available.

Search seems slow

For large databases, wait a moment after typing for results to load. Using more specific terms (full name instead of first letter) helps.
