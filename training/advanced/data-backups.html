Module 10

Data & Backups

4 min read

Your business data is valuable. Understanding how backups work gives you peace of mind and helps you recover from unexpected issues.

Automatic Backups

The platform automatically backs up all data:

  • Database backups — Full database backups run daily
  • Point-in-time recovery — The database can be restored to any point in the last 30 days
  • Redundancy — Data is stored across multiple geographic regions

You don't need to do anything. Backups happen automatically. This section is for your awareness so you understand how your data is protected.

Manual Data Exports

For extra safety or for moving data to another system, you can manually export:

  • Client data — Export from the Client Database
  • Financial reports — Export from Accounting Reports
  • Full data export — Request a complete data export from Settings → Data Management

Data Recovery

If you accidentally delete something important:

  1. Check the Trash
    Deleted client records go to the trash first (30-day retention). Restore from there.
  2. Contact Support
    For other data recovery needs, contact your system administrator. They can restore from backups.

Export files contain sensitive data. Store exports securely and delete them when no longer needed. Never email unencrypted data exports.
