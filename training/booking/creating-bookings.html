Module 6

Creating Bookings

5 min read

Create bookings by selecting a client, service, provider, and available time slot. The system prevents double-booking and sends automatic confirmations.

Step-by-Step Booking

  1. Click + New Booking
    From the calendar view, click New Booking. Or double-click a time slot on the calendar.
  2. Select the Client
    Search for the client by name, phone, or client ID. If they're new, create their record first.
  3. Choose a Service
    Select the service from the catalog. The duration is automatically set based on the service definition.
  4. Select a Provider
    Choose the service provider. Only available providers (those with open time slots) are shown.
  5. Pick Date and Time
    Select the date and available time slot. The system shows only open slots based on the provider's schedule.
  6. Add Notes (Optional)
    Add any special notes (e.g., "Client prefers room 2" or "First visit — needs intake form").
  7. Confirm Booking
    Review the details and click Book Appointment. The system saves the booking and sends a confirmation to the client.

No double booking. The system won't allow you to book two clients with the same provider at the same time. If a conflict exists, choose a different time or provider.

Recurring Bookings

For clients with regular schedules:

  1. Check "Recurring"
    In the booking form, toggle the Recurring option.
  2. Set Frequency
    Choose Weekly, Bi-weekly, or Monthly.
  3. Set End Date
    Choose how many sessions or an end date.
