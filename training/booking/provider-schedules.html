Module 6

Provider Schedules

4 min read

Provider schedules define when each service provider is available for bookings. Proper setup prevents overbooking and ensures clients see accurate availability.

Setting Working Hours

  1. Go to Settings → Providers
    Navigate to the provider management area.
  2. Select a Provider
    Click on the provider whose schedule you want to set.
  3. Set Weekly Hours
    For each day of the week, set the start and end times. Leave a day blank if the provider doesn't work that day.
  4. Add Breaks
    Click + Add Break to block out lunch or other break times within a working day.
  5. Save
    Click Save Schedule. The booking calendar immediately reflects these hours.

Blocking Time Off

  1. Click Add Time Off
    From the provider's schedule page, click Add Time Off.
  2. Select Dates
    Choose the start and end dates of the time off.
  3. Add a Reason (Optional)
    Enter "Vacation", "Training", "Personal", etc.
  4. Save
    Those dates are blocked — no bookings can be made for this provider during that period.

Check existing bookings first. If the provider already has bookings during the time-off period, you'll need to reschedule them before blocking the time.
