Module 4

Handling Walk-Ins

4 min read

Walk-in clients arrive without a prior appointment. You'll need to check availability, register them if they're new, and get them checked in for service.

Walk-In Workflow

  1. Greet the Client
    Welcome them and ask what service they're looking for today.
  2. Check Availability
    Look at the booking calendar to see if a provider is available for the requested service. If fully booked, offer the next available slot.
  3. Search for Existing Record
    Search the client database by name or phone to see if they already have a profile.
  4. Register if New
    If they're not in the system, create a quick new client record with their name and contact info. See Adding New Clients.
  5. Check Them In
    Use the New Walk-In button on the dashboard to check them in as a walk-in. Select the service and provider.

Speed matters. Walk-ins expect quick service. Have the new client form ready to go and keep the registration to essential fields only — you can complete the full profile later.

When No Availability

If no providers are available right now:

  • Offer the next available time slot today
  • Offer to book them for a future date
  • Add them to a waitlist if one is available
  • Register them as a new client so they're in the system for next time
