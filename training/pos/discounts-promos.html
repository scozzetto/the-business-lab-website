Module 5

Discounts & Promotions

4 min read

Apply percentage or dollar-amount discounts during POS transactions. You can also enter promotional codes for special offers.

Applying a Discount

  1. Add Items to Cart
    Add the items the client is purchasing.
  2. Click Discount
    Click the Discount button below the cart total.
  3. Choose Discount Type
    Select Percentage (e.g., 10% off) or Fixed Amount (e.g., $5 off).
  4. Enter the Value
    Type the discount percentage or dollar amount.
  5. Apply
    Click Apply Discount. The cart total updates to reflect the discount.

Promotional Codes

  1. Click Promo Code
    Click the Promo Code field in the cart area.
  2. Enter the Code
    Type the promotional code exactly as provided (codes are not case-sensitive).
  3. Apply
    Click Apply. If valid, the discount is applied automatically based on the promo's rules.

Manager approval may be required for discounts above a certain threshold (e.g., more than 20%). The system will prompt for manager override if needed.
