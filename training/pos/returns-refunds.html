Module 5

Returns & Refunds

5 min read

Returns, refunds, and exchanges are part of daily retail. This lesson covers how to process each type correctly.

Processing a Return

  1. Click Returns
    On the POS terminal, click the Returns button to enter return mode.
  2. Find the Original Transaction
    Search by receipt number, client name, or date. Select the original sale.
  3. Select Items to Return
    Check the items being returned. You can return individual items or the entire order.
  4. Select Refund Method
    Choose how to refund: original payment method (card refund), cash, or store credit.
  5. Process the Return
    Click Process Return. Card refunds go back to the original card. Cash refunds come from the drawer.

Card refunds take 5-10 business days to appear on the client's statement. Inform the client of this timeline.

Processing an Exchange

An exchange is a return + a new sale in one step:

  1. Process the Return
    Follow the return steps above, but select "Exchange" instead of "Refund."
  2. Add the New Item
    Scan or select the replacement item. The system calculates the price difference.
  3. Settle the Difference
    If the new item costs more, collect the difference. If it costs less, refund the difference.

Voiding a Transaction

If a transaction was just completed (same day) and needs to be completely reversed, use Void instead of Return. Voids cancel the transaction as if it never happened.

Check your return policy. Follow your business's return policy regarding time limits, condition requirements, and receipt requirements. The system may enforce these automatically.
