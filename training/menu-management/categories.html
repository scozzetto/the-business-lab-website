Categories & Organization

4 min read

Categories group related items together and control how they appear in the POS grid, catalog listings, and reports.

Creating a Category

  1. Go to Catalog → Categories
    Click the Categories tab in the catalog manager.
  2. Click + New Category
    Open the category creation form.
  3. Enter Details
    Fill in the Name, optional Description, and optional Color (used in the POS grid for visual grouping).
  4. Set Sort Order
    Enter a number to control where this category appears in the list. Lower numbers appear first.
  5. Save
    Click Save Category.

Assigning Items to Categories

Each item belongs to one category. Set it when creating the item, or change it later by editing the item and selecting a different category from the dropdown.

Category Best Practices

  • Keep category names short and clear (5 words or fewer)
  • Use 5-10 categories — too many defeats the purpose of organization
  • Consider how staff will browse the POS grid when naming categories
  • Use colors to make visual scanning faster in the POS

Example categories: Beverages, Food, Wellness Products, Services, Memberships, Packages, Gift Cards.
