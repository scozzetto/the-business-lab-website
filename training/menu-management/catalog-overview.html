Catalog Overview

4 min read

The catalog manager is where you define everything your business offers. Items you add here appear in the POS, booking system, and billing forms.

Accessing the Catalog

  1. Click Catalog in the Sidebar
    Navigate to the Catalog section from the main sidebar.
  2. Browse the Item List
    Items are displayed in a searchable, filterable list organized by category.

Item Types

TypeDescriptionExample
ProductPhysical items with inventorySkincare products, supplements
ServiceTime-based offeringsHaircut, massage, consultation
PackageBundled products/services"New Client Package" (3 sessions)
MembershipRecurring subscriptionMonthly wellness plan

List View vs. Grid View

Toggle between list view (detailed table) and grid view (visual cards with images) using the view toggle buttons. List view is better for management; grid view is better for visual browsing.

Tip: Use the search bar at the top to quickly find any item by name, SKU, or category.
