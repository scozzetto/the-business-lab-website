Module 8

Exporting Reports

3 min read

Export any report as a CSV file (for spreadsheets) or PDF (for printing and sharing). You can also email reports directly to stakeholders.

Exporting a Report

  1. Generate the Report
    Set your date range and filters to get the data you need.
  2. Click Export
    Click the Export button in the report toolbar.
  3. Choose Format
    Select CSV (for Excel/Sheets analysis) or PDF (for print/share).
  4. Download
    The file downloads to your computer.

Emailing Reports

  1. Click Email Report
    Next to the Export button, click Email Report.
  2. Enter Recipients
    Type one or more email addresses.
  3. Add a Note (Optional)
    Include a message with the report.
  4. Send
    The report is sent as a PDF attachment.

Format Comparison

FormatBest For
CSVSpreadsheet analysis, custom calculations, importing into accounting software
PDFPrinting, emailing to stakeholders, archival

Handle financial reports securely. Don't leave exported reports on shared computers. Delete files when no longer needed.
